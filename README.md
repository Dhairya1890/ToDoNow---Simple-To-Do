<h1>ToDoNow - A Simple Todo with Flask </h1>

<h2> Tech Stack </h2>
<ul>
<li> Python (Flask) v3.13 </li>
<li> jinja2 </li>
<li> HTML/CSS </li>
<li> SQLite </li>
</ul>

<h2> Current Features </h2>
<ul>
<li> Add, Update or Delete tasks </li>
<li> Task Cards </li>
</ul>

<h2> Further Implementation </h2>
<ul>
<li> Reminder, Timer etc... </li>
<li> Authentication - Guest and Admin </li>
<li> Cloud Storage </li>
<li> Linking tasks with other tasks </li>
<li> AI Summarizer and action planner </li>
</ul>

<h3> Installation </h3>

<p> 1. Clone Repository </p>
`git clone "https://github.com/Dhairya1890/ToDoNow---Simple-To-Do"`
<br>
<p> 2. Create a Virtual Environment </p>
`python -m venv venv`
<br>
<p> 3. Install Requirements </p>
`pip install -r requirements.txt`
<br>
<p> 4. Start the server </p>
`python app.py`
<br>
